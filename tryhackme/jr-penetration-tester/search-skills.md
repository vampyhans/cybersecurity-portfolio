# TryHackMe - Search Skills

## Overview

The Search Skills room focused on developing effective information-gathering and research techniques for cybersecurity. The room demonstrated how to efficiently search for technical information, vulnerabilities, documentation, and security-related resources using various online platforms.

* Open Source Intelligence (OSINT)
* Vulnerability Research

* Search Engine Techniques

## Tools and Resources

## Key Concepts Learned

### Effective Searching

Cybersecurity professionals frequently rely on search engines and specialized resources to gather information quickly and accurately.

### Vulnerability Research

Public vulnerability identifiers (CVEs) can be used to locate:

* Proof-of-Concept (PoC) code
* Technical analyses
* Mitigation recommendations
* Community discussions

### GitHub as a Security Resource

GitHub can provide:

* Security research
* Exploit demonstrations
* Scanner tools
* Technical writeups

However, repositories should always be reviewed carefully before executing any code.

## Practical Exercise

The practical exercise involved investigating a GitHub repository associated with a fictional vulnerability and reviewing available documentation to identify relevant exploitation resources.

## Key Takeaways

* Strong search skills are essential for cybersecurity professionals.
* GitHub is a valuable resource for security research.
* Technical documentation often provides critical context for vulnerabilities.
* Public Proof-of-Concept code should always be reviewed before use.
* Research skills can significantly improve incident response and vulnerability assessment activities.
